# AI ATS Resume Optimizer

## Overview

An AI-powered system that:
- Uploads resumes
- Matches them with job descriptions
- Calculates ATS score
- Optimizes resumes using LLMs
- Generates ATS-friendly resumes

## Tech Stack

- Python
- Streamlit
- NLP
- scikit-learn
- Ollama
- Llama 3
- ChromaDB

## Features

- Resume Upload
- PDF/DOCX Parsing
- ATS Score Calculation
- JD Matching
- AI Resume Optimization
- Manual Resume Editing
- PDF Export
